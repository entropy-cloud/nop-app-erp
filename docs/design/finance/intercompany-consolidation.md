# 多公司合并抵消

## 目的

说明集团多公司场景下的内部交易抵消、合并报表生成机制。参考 iDempiere 多公司架构和 Metasfresh 合并报表设计。

## 业务场景

### 典型应用场景

| 场景 | 说明 | 示例 |
|------|------|------|
| 集团合并报表 | 母子公司合并财务报表 | 年度集团合并资产负债表、利润表 |
| 内部交易抵消 | 消除集团内部交易影响 | 内部销售收入 vs 内部采购成本抵消 |
| 内部往来抵消 | 消除集团内部应收应付 | A公司应收B公司 vs B公司应付A公司抵消 |
| 内部利润抵消 | 消除未实现内部利润 | 存货中包含的内部加价利润抵消 |

### 合并范围确定

```
合并范围确定
        │
        ├─► 控制权评估
        │      ├─ 持股比例 > 50%
        │      ├─ 实质控制权（协议/任命/重大影响）
        │      └─ 特殊目的实体（SPE）
        │
        ├─► 合并层级
        │      ├─ 母公司 + 全资子公司
        │      ├─ 母公司 + 控股子公司
        │      └─ 母公司 + 联营/合营（权益法）
        │
        └─► 合并期间
               ├─ 全年合并
               └─ 购买日起合并（年中并购）
```

## 核心业务对象

| 对象 | 业务含义 | 头/行 |
|------|----------|-------|
| 合并方案（ConsolidationScheme） | 集团合并参数配置 | 头 |
| 合并范围（ConsolidationScope） | 纳入合并的公司清单 | 头/行（公司列表） |
| 内部交易对账（IntercompanyReconciliation） | 集团内部交易对账 | 头/行 |
| 合并调整凭证（ConsolidationAdjustment） | 合并抵消分录 | 头/行 |
| 合并报表（ConsolidationReport） | 合并后财务报表 | 头/行 |

## 内部交易抵消机制

### 内部交易类型

| 交易类型 | 抵消分录 | 说明 |
|----------|----------|------|
| 内部销售/采购 | 借：营业收入 / 贷：营业成本 | 抵消内部销售收入与成本 |
| 内部应收/应付 | 借：应付账款 / 贷：应收账款 | 抵消内部往来余额 |
| 内部投资/权益 | 借：实收资本 / 贷：长期股权投资 | 抵消内部投资与权益 |
| 内部利润 | 借：营业收入 / 贷：营业成本 + 存货 | 抵消存货中未实现内部利润 |
| 内部股利 | 借：投资收益 / 贷：利润分配-应付股利 | 抵消内部股利分配 |

### 抵消分录生成流程

```
合并方案启动
        │
        ├─► 收集各子公司财务数据
        │      ├─ 读取各公司 ErpFinVoucherLine
        │      └─ 按合并科目映射转换
        │
        ├─► 内部交易对账
        │      ├─ A公司销售 → B公司采购
        │      ├─ 核对金额、期间、科目
        │      └─ 标记差异（未达账项）
        │
        ├─► 生成抵消分录
        │      ├─ 内部销售/采购抵消
        │      ├─ 内部往来抵消
        │      ├─ 内部投资/权益抵消
        │      └─ 内部利润抵消
        │
        ├─► 合并调整
        │      ├─ 公允价值调整
        │      ├─ 会计政策统一
        │      └─ 外币报表折算
        │
        └─► 生成合并报表
               ├─ 合并资产负债表
               ├─ 合并利润表
               └─ 合并现金流量表
```

## 科目映射机制

### 合并科目映射

各子公司科目 → 集团统一合并科目的映射：

| 子公司科目 | 合并科目 | 说明 |
|------------|----------|------|
| 1001 银行存款 | 1001 银行存款 | 直接映射 |
| 1122 应收账款 | 1122 应收账款（其中：内部往来） | 标记内部往来 |
| 2202 应付账款 | 2202 应付账款（其中：内部往来） | 标记内部往来 |
| 6001 主营业务收入 | 6001 营业收入（其中：内部交易） | 标记内部交易 |

### 合并调整类型

| 调整类型 | 说明 | 示例 |
|----------|------|------|
| 公允价值调整 | 资产按公允价值入账 | 收购时固定资产增值 |
| 会计政策统一 | 统一折旧方法、收入确认等 | A公司直线法，B公司加速折旧 |
| 外币报表折算 | 外币报表折算为母公司记账本位币 | 美国子公司美元报表折算为人民币 |
| 股权调整 | 非全资子公司少数股东权益 | 持股80%，20%为少数股东权益 |

## 实现要点

### 数据采集

```
合并数据采集
        │
        ├─► 读取各子公司 ErpFinGlBalance
        │      ├─ 按 acctSchemaId + periodId + subjectCode 汇总
        │      └─ 转换为合并科目
        │
        ├─► 读取内部交易明细
        │      ├─ 按 intercompany 标记筛选
        │      └─ 双向核对（A公司应收 vs B公司应付）
        │
        └─► 读取汇率
               ├─ 历史汇率（投资日）
               ├─ 期末汇率（资产负债表日）
               └─ 平均汇率（利润表）
```

### 抵消分录模板

| 抵消场景 | 借方科目 | 贷方科目 | 金额来源 |
|----------|----------|----------|----------|
| 内部销售 | 6001 营业收入 | 6401 营业成本 | A公司销售收入 = B公司采购成本 |
| 内部应收应付 | 2202 应付账款 | 1122 应收账款 | A公司应收余额 = B公司应付余额 |
| 内部存货利润 | 6001 营业收入 | 6401 营业成本 + 1403 存货 | 未实现利润 = 存货价值 × 内部加价率 |
| 内部投资 | 4001 实收资本 | 1511 长期股权投资 | 母公司投资额 = 子公司实收资本 × 持股比例 |

## 配置项

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| `erp-fin.consolidation-currency` | CNY | 合并报表记账本位币 |
| `erp-fin.consolidation-method` | FULL | 合并方法（FULL/PARTIAL/EQUITY） |
| `erp-fin.intercompany-tolerance` | 0.01 | 内部交易对账容差（金额差异） |
| `erp-fin.consolidation-schedule` | MONTHLY | 合并频率（MONTHLY/QUARTERLY/YEARLY） |
