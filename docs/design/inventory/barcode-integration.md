# 条码/PDA 集成（Barcode Integration）

## 目的

设计条码扫描与 PDA 移动终端在 ERP 各环节的应用：收料、上架、拣货、发货、盘点、领料、质检。条码是连接实物与系统数据的桥梁，PDA 是现场作业的实时交互终端。

## 边界

- 本模块负责：条码规则配置、条码标签打印（SKU/批次/序列号/库位）、PDA 移动终端操作界面（收货/上架/拣货/发货/盘点/移库）、与 inventory/manufacturing/quality 各域的实时交互。
- 本模块不负责：RFID 硬件驱动（适配层由 PDA 厂商 SDK 提供）；条码打印硬件驱动（驱动由打印机厂商提供）。

## 设计依据

> 参考 AureusERP `barcode` 插件（12 PHP 文件）+ WMES PDA 扫码方案。

## 条码体系

### 支持的条码类型

| 条码类型 | 编码规则 | 用途 |
|---------|---------|------|
| SKU 条码 | EAN-13 / UPC-A / 内部编码 | 商品唯一标识，收/发/盘通用 |
| 批次条码 | 前缀(2) + 物料编码 + 批次号 + 日期 | 批次追溯，收货时打印 |
| 序列号条码 | 序列号规则（128码） | 单品追踪，出/入库扫码 |
| 库位条码 | 仓库编码 + 区域 + 排 + 列 + 层 | 上架/拣货定位 |
| 单据条码 | 单号（128码） | 快速调出业务单据 |
| 托盘/容器码 | 生成唯一码 | 批量移动/装车 |

### 条码生成规则

通过 nop-sys 序列号引擎配置（`ErpSysDocNumberRule`），按 `billType` + `orgId` 生成条码字符串：
- `{SKU_CODE}` — 物料编码
- `{BATCH_NO}` — 批次号
- `{SERIAL_NO}` — 序列号
- `{LOCATION_CODE}` — 库位号
- `{DOC_CODE}` — 单据号

## PDA 功能场景

### 1. 收货（Purchase Receive）

```
PDA 扫码收货流程：
  1. 采购员扫描采购单条码 → PDA 显示待收明细
  2. 扫描 SKU + 批次（逐件或批量输入数量）
  3. 扫描目标库位（自动推荐，可改）
  4. 提交 → 调用 IErpInvStockMoveBiz 生成入库移动单
```

### 2. 上架（Putaway）

```
PDA 上架流程：
  1. 扫描收货后的托盘/容器码
  2. PDA 推荐上架库位（按 ABC 分类/周转率）
  3. 扫描目标库位确认
  4. 提交 → 更新 StockBalance 库位
```

### 3. 拣货（Picking）

```
PDA 拣货流程：
  1. 扫描出库单/波次码 → PDA 显示拣货清单
  2. 逐件/逐箱扫描 SKU + 批次
  3. 扫描来源库位（校验 FIFO/FEFO）
  4. 提交 → 调用 IErpInvStockMoveBiz 生成出库移动单
```

### 4. 发货（Shipping）

```
PDA 发货流程：
  1. 扫描装车单/运单号
  2. 逐件扫描已拣货物确认装车
  3. 打印发货标签/运单
  4. 提交 → 出库单状态→DONE
```

### 5. 盘点（Stock Count）

```
PDA 盘点流程：
  1. 选择/扫描盘点单 → PDA 加载盘点列表
  2. 扫描 SKU + 库位 → 输入实盘数量
  3. 差异实时显示
  4. 提交 → 生成盘点差异移动单
```

### 6. 领料/退料（Manufacturing）

```
PDA 领料流程：
  1. 扫描工单条码 → PDA 显示 BOM 物料清单
  2. 扫描 SKU + 批次 → 扫描库位
  3. 提交 → 生成领料出库移动单

PDA 退料流程：
  1. 扫描工单条码 → 扫描退料 SKU
  2. 输入数量 → 扫描目标库位
  3. 提交 → 生成退料入库移动单
```

### 7. 质检采样（Quality）

```
PDA 质检流程：
  1. 扫描质检单条码 → 加载检验项目
  2. 录入实测值（直接输入或扫描）
  3. 判定 ACCEPTED/REJECTED
  4. 提交 → 回写质检结果
```

## 跨域协作

| 对端 | 协作方式 |
|------|---------|
| inventory/StockMove | PDA 收货/发货/盘点调用 IErpInvStockMoveBiz |
| inventory/StockCount | PDA 盘点提交差异 |
| manufacturing/WorkOrder | PDA 领料/退料/完工扫描 |
| quality/Inspection | PDA 质检结果提交 |

## 配置点

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| `erp-inv.barcode-enabled` | true | 是否启用条码扫描功能 |
| `erp-inv.barcode-sku-format` | EAN13 | SKU 条码格式 |
| `erp-inv.barcode-auto-print-on-receive` | true | 收货时是否自动打印批次/序列号标签 |
| `erp-inv.pda-require-location-scan` | true | PDA 操作是否强制扫描库位 |

## 反模式警示

- ⛔ **条码规则硬编码**——不同行业条码格式不同（零售 EAN13 / 物流 128码 / 医药监管码），必须通过配置管理。
- ⛔ **PDA 同步阻塞**——PDA 操作应支持离线队列（先操作后同步），不依赖实时网络。
- ⛔ **条码替代业务单据号**——条码是实物标识符，业务单据号是系统标识符，两者共存。
