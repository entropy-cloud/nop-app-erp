# Audits Index

Use this directory for audit methodology and audit records.

Start with `00-audit-execution-guide.md`.

Independent draft review and closure audit are mandatory for created plans. Closure audit must come from an independent subagent or reviewer, not the implementing agent alone. Store audit records here when they are non-trivial, disputed, or useful for future sessions; otherwise keep draft-review evidence in the plan or daily log and closure evidence in the plan `## Closure` section.

Recommended filenames:

- `2026-05-21-1100-document-audit-user-management.md`
- `2026-05-21-1100-draft-review-checkout-flow.md`
- `2026-05-21-1100-closure-audit-order-list.md`
