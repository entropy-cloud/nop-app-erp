## 关键模块分析

### 1. 制造管理（manufacturing：110 PHP 文件）

**源码目录**：`plugins/webkul/manufacturing/src/`

**核心能力**：
- BOM（Bill of Materials），支持多级 BOM 展开
- Manufacturing Order（生产工单）—— 创建→确认→执行→完成
- Work Order（工序工单）
- Work Center（工作中心）—— 产能建模（OEE/效率/产能）
- Operations（工序定义）
- Raw Material Requirements（原材料需求计算）
- Finished Product Receipt（成品入库）

**对 nop 的借鉴**：Work Center 产能模型 + Work Order 工序级执行，是对 CRMD 的完整实现。

### 2. 库存/仓库管理（inventories：294 PHP 文件）—— **覆盖高级 WMS 缺口**

**源码目录**：`plugins/webkul/inventories/src/`

**能力**：
- 多仓库/多库位管理
- 库存移动
- 入库/出库单
- 盘点
- 库位转移
- **仓储操作配置**（Operation Types 可定义入库/出库/转移等不同作业）

### 3. 人力资源 3 件套（employees + recruitments + time-off）—— **覆盖 P2 HRMS 缺口**

**员工管理**（91 PHP 文件）：
- Employee CRUD
- 部门/职位管理
- 合同信息

**招聘管理**（113 PHP 文件）：
- 应聘者追踪（Applicant Tracking）
- 职位发布
- 面试流程

**休假管理**（103 PHP 文件）：
- 请假申请 → 审批
- 假期类型配置
- 可用假期余额计算

### 4. 支持服务（support：148 PHP 文件）—— **覆盖 P2 售后服务缺口**

**源码目录**：`plugins/webkul/support/src/`

**能力**：
- Ticket 工单系统
- FAQ/知识库
- SLA 管理

### 5. 财务核算（accounting + accounts：514 PHP 文件）

**accounting（209 PHP 文件）**：
- 总账
- 凭证管理
- 试算平衡表

**accounts（305 PHP 文件）**：
- 科目表
- 应收/应付
- 银行对账

### 6. 项目管理（projects：84 PHP 文件）

- 项目创建/阶段/里程碑
- 任务分配与追踪
- 项目协作（chatter 集成）

## 对 nop-app-erp 的借鉴建议

### 需覆盖的功能

| 功能 | AureusERP 插件 | 本项目当前状态 | 纠正方向 |
|------|---------------|--------------|---------|
| 员工管理 + 招聘 | employees + recruitments | P2 排除 → 纳入设计 | Employee 主数据 + 招聘流程 |
| 休假管理 | time-off（103 files） | P2 排除 → 纳入设计 | 请假/审批/余额管理 |
| 客服工单 | support（148 files） | P2 排除 → 纳入设计 | Ticket + FAQ + SLA |
| 条码/PDA 支持 | barcode | 未覆盖 → 纳入设计 | 条码扫描入库/出库 |

### 项目特色

1. **插件化架构**：每个插件独立 Composer 包，可安装/卸载，这与 nop 的 Delta 定制思想一致。
2. **Laravel 13 + FilamentPHP 5**：使用 PHP 最新技术栈，但完整度不如 Java 系的 Axelor。
3. **MIT License**：完全开源无附加限制。
4. **工时表（timesheets）**：简单工时记录，与 projects 集成。
